# The Grand お試し部屋 仕様書
最終更新: 2026-04-18

## 目的
`The Grand お試し部屋` は、1人で自分・味方・敵を配置し、HP やコストを変えながら、各キャラの `小技 / 中技 / 大技` をすぐ検証するための部屋です。

## できること
- 自分キャラの即時変更
- 味方1 / 味方2 の即時変更
- 敵キャラの即時変更
- 自分 / 味方1 / 味方2 / 敵 の盤面配置
- 現在の操作ユニットの切替
- 自分 / 味方の HP 管理
- 敵の HP 管理
- 自分 / 味方のコスト管理
- `小技 / 中技 / 大技` の切り替え
- 検証ログの確認
- 技の説明確認
- ハムスター召喚後の操作切替

## 画面で確認できる情報
技を選ぶと、右側に次を表示します。
- 移動種別
- 攻撃力
- 説明
- 効果範囲
- 図の見方
- 現在の実装

## 味方を置く理由
- `サウザンド・アイ` の視界共有確認
- `聖女の祈り` のランダム対象確認
- `再配置` や `再構成` の将来デバッグ準備
- `リフレクト` と敵行動の相互確認

## 現在の実装方針
- ラボでは、できるだけ本仕様に沿って処理します。
- ただし、味方複数人や残りターン編集が必要な技は、ラボでは `説明確認用の近似` を含みます。
- 本編に戻す前に、ここで挙動と強さを詰める前提です。

## 注意
- ラボは1人検証用です。

- 本編完全一致ではない技は、`現在の実装` 欄とこの仕様書に明記します。
